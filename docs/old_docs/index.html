研究進度報告

專案名稱：透過 Persona Vector Merge 提升大型語言模型創造力

1. 專案核心理念

現有的 Multi-Agent (如 LLM-Discussion 框架) 方法雖然能提升創造力，但其依賴多次、輪流的線上推理，面臨成本高昂、推理延遲巨大的瓶頸。本研究提出一種創新的 Persona Vector Merge 機制，將「角色模擬」的過程從「線上互動」轉變為「離線表示」。我們透過**離線 (Offline)** 計算並融合多個角色的 Persona Vector，創造出一個單一的、富含多維視角的複合向量 (Composite Vector)。在推理時，這個複合向量能一次性地引導單一模型，使其在一次前向傳播中，就能模擬出多角色腦力激盪的效果。這代表了從「互動式模擬 (Interactive Simulation)」到「表示式引導 (Representational Steering)」的範式轉移，旨在達成 「降本增效」的雙重目標。

2. Benchmarks and Evaluation Metrics

2.1 評估指標 (Metrics)

We employ the metrics established by the Torrance Tests of Creative Thinking (TTCT) (Torrance, 1966), recognized for their robustness in measuring creative capabilities and are commonly applied in assessing human creativity (Cramond et al., 2005). The TTCT framework comprises four distinct metrics:

  • Originality: Considering novelty responses, not familiar and unusual, but relevant.
  • Elaboration: The amount of details used to extend a response.
  • Fluency: The number of relevant responses.
  • Flexibility: A variety of categories or shifts in responses.

2.2 基準測試任務 (Benchmarks)

Benchmark Description Sample Task
AUT
(Alternative Use Task)		 This benchmark requires participants to come up with as many creative uses as possible for a simple object. What are some creative uses for a fork?
INSTANCES This benchmark requires participants to list as many creative answers as possible that fit a specific criterion. Name all the round things you can think of.
SIMILARITIES This benchmark asks participants to creatively explore similarities between two things. Tell me all the ways in which a book and a magazine are alike.
SCIENTIFIC This benchmark asks participants to generate innovative ideas, envision hypothetical scenarios, propose improvements, and design experiments, all scientifically. Please think of as many possible improvements as you can to a regular bicycle, making it more interesting, more useful, and more beautiful.

4. 目前實驗成果亮點

雙重勝利

在成本 (Cost) 與效能 (Performance) 上，我們的方法均顯著優於現有的 Multi-Agent Baseline。

小模型超越大模型

實驗證明，我們使用的小模型 (7B/8B 等級) 在整合 Persona Vector 後，其創造力指標 (特別是 Originality 和 Elaboration) 能穩定勝過數十倍大的模型 (如 Llama-405B, GPT-4o-mini) 在 Multi-Agent 框架下的表現。

5. 現況總結與分析

目前專案進展順利，核心假設已由初步實驗獲得強力支持。數據顯示我們的方法在關鍵指標上不僅有效，甚至超越了資源消耗更大的 SOTA 方法。當前的重點在於鞏固實驗的嚴謹性，並深化對方法內在機制的理解。

  • 優勢 (Strengths):
    • 核心想法創新，實用價值高，提出「表示式引導」的新範式。
    • 初步數據非常強勁，故事線清晰 (e.g., 在 AUT 任務中，8B 模型效果超越 405B 模型)。
    • 實驗控制良好 (如 Prompt 設計公平)，能有效突顯方法本身的優越性。
    • 已驗證方法的魯棒性 (Baseline 方法在小模型上效果顯著衰退，而本方法表現穩定)。
  • 待補強處 (Weaknesses):
    • 目前與 Baseline 的比較存在樣本數不對等的問題 (100 vs 10)，這是確保結論無可辯駁、應對頂會嚴格審查的關鍵一步。

6. 待辦事項與未來方向 (目標：ICLR / ARR October)

1 第一優先級：補全核心實驗

  • (最緊急) 為所有 Baseline 方法 (Multi-Agent Discussion, Vanilla Single Agent) 補齊 N=100 的實驗數據，確保比較的絕對公平性。

2 第二優先級：構建論文核心內容

  • 設計論文的「主結果表」，清晰呈現本方法在**四個任務**上，與**多種 Baseline** (包含不同大小模型) 的全面比較。

3 第三優先級：深化分析 (ICLR 關注點)

  • 消融實驗 (Ablation Study): 深入分析不同設計選擇的影響。例如：比較 [‘科學家’, ‘藝術家’] vs [‘科學家’, ‘工程師’, ‘哲學家’] 兩種組合，探討 persona 的數量與「語義多樣性」如何影響創造力分數。同時，對比「向量平均法」與「加權平均法」，找出最優的融合策略。
  • 向量空間分析:
    • 可視化分析：使用 t-SNE / UMAP 進行可視化，直觀展示融合後向量在語義空間中的位置。例如：在 t-SNE 圖上，我們預期「科學家」和「工程師」的向量會比較接近，而「藝術家」的向量會較遠。我們融合後的向量「科藝工程師」應該會出現在這三個點構成的三角形區域內，直觀證明融合的有效性。
    • 語義探針 (Semantic Probing)：設計探針任務，證明融合後的向量確實繼承了多個 Persona 的特性。例如：給定 prompt “描述水的未來”，我們期望「科學家」向量會生成關於水資源淨化、相變儲能的內容（高邏輯性）；「藝術家」向量會生成關於眼淚形成的海洋、記憶組成的雨滴等（高想像力）；而我們融合後的向量，理想上應能生成兼具兩者特質的文本，例如「一種利用量子糾纏實現的即時水分子重組技術，能將沙漠中的濕氣詩意地凝結成可飲用的晨露」。
