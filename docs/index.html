研究進度報告

專案名稱：透過 Persona Vector Merge 提升大型語言模型創造力

專案核心理念

現有的 Multi-Agent (如 LLM-Discussion 框架) 方法雖然能提升創造力，但面臨成本高昂、推理延遲巨大的瓶頸。本研究提出一種創新的 PersonaAPI 框架，透過**離線 (Offline)** 計算並融合多個角色的 Persona Vector，讓單一模型在一次前向傳播中，就能模擬出多角色腦力激盪的效果，旨在達成 「降本增效」的雙重目標。

目前實驗成果亮點

雙重勝利

在成本 (Cost) 與效能 (Performance) 上，我們的方法均顯著優於現有的 Multi-Agent Baseline。

小模型超越大模型

實驗證明，我們使用的小模型 (7B/8B 等級) 在整合 Persona Vector 後，其創造力指標 (特別是 Originality 和 Elaboration) 能穩定勝過數十倍大的模型 (如 Llama-405B, GPT-4o-mini) 在 Multi-Agent 框架下的表現。

通用性驗證

研究已涵蓋四種不同類型的任務 (AUT, scientific test, similarity, instances)，初步證明了方法的通用性，而不僅限於單一任務。

現況總結與分析

  • 優勢 (Strengths):
    • 核心想法創新，實用價值高。
    • 初步數據非常強勁，故事線清晰。
    • 實驗控制良好 (如 Prompt 設計公平)，能有效突顯方法本身的優越性。
    • 已驗證方法的魯棒性，在小模型上效果依然出色，反觀 Baseline 方法則出現性能下降。
  • 待補強處 (Weaknesses):
    • 目前與 Baseline 的比較存在樣本數不對等的問題 (100 vs 10)，這是實驗嚴謹性的最大挑戰。

待辦事項與未來方向 (目標：ICLR)

1 第一優先級：補全核心實驗

  • (最緊急) 為所有 Baseline 方法 (Multi-Agent Discussion, Vanilla Single Agent) 補齊 N=100 的實驗數據，確保比較的絕對公平性。

2 第二優先級：構建論文核心內容

  • 設計論文的「主結果表」，清晰呈現本方法在**四個任務**上，與**多種 Baseline** (包含不同大小模型) 的全面比較。

3 第三優先級：深化分析 (ICLR 關注點)

  • 消融實驗 (Ablation Study): 分析不同 Vector Merge 策略、Persona 數量與組合對結果的影響。
  • 向量空間分析:
    • 使用 t-SNE / UMAP 進行可視化，直觀展示融合後向量在語義空間中的位置。
    • 設計語義探針，證明融合後的向量確實繼承了多個 Persona 的特性。

4 論文敘事架構

  • 確立論文故事線：從現有方法的三大痛點 (成本、延遲、兼容性) 出發，引出我們的解決方案，並透過全面實驗和深入分析來證明其優越性。
